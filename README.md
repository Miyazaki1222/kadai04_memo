# ①課題名
日本地図にステータス記録

## ②課題内容（どんな作品か）
- 都道府県のステータスを選択すると該当箇所の色が変化
- ステータスごとに点数を割当て、合計点数を表示

## ③アプリのデプロイURL

## ④アプリのログイン用IDまたはPassword（ある場合）
- ID: なし
- PW: なし

## ⑤工夫した点・こだわった点
- ラジオボタンのテーブルが47都道府県を表示させると縦に長くなるためスクロールで表示
- 都道府県ごとに地図とリンクする1~47の番号を割当て
- ステータスごとに地図の色や点数とリンクする0~4の番号を割当て

## ⑥難しかった点・次回トライしたいこと（又は機能）
- 日本地図のsvgファイルは、参考URLを引用。パスで範囲を選択する必要があることまで理解
- 地図をクリックすると画像ファイルをアップロードできるようにしたが、表示方法など工夫できていない
